# 抽獎程式 (Lucky Draw App)

這是一個使用 HTML、CSS（Bootstrap）、JavaScript 製作的單頁式抽獎應用程式，支援 Edge 及 Chrome 瀏覽器。

## 功能特色
- 支援獎項名稱與數量輸入格式檢查
- 支援參加者名單格式檢查
- 亂數抽獎且不重複中獎
- 每抽出一位可確認是否在場，不在場可重抽並剃除
- 抽獎動畫加速與過場效果
- SPA 架構，無須額外安裝軟體
- 支援 Edge 135+、Chrome 135+

## 使用方式
1. 下載或 clone 此專案
2. 直接開啟 `index.html` 即可使用

## 專案結構
- `index.html`：主頁面
- `styles.css`：自訂樣式
- `app.js`：主要功能腳本
